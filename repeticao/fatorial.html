Ajuda - Como Criar Fluxos

Tipos de Blocos

  • Início: Ponto de partida do fluxo.
  • Entrada: Define variáveis e seus valores iniciais.
  • Decisão: Usado para representar condições.
  • Processo: Realiza operações com as variáveis.
  • Saída: Representa a referência de uma variável ou resultado.
  • Fim: Ponto final do fluxo.

Como Criar um Fluxo

  1. Arraste os blocos da paleta para o editor.
  2. Conecte os blocos arrastando os pontos de conexão para definir a sequência lógica.
  3. Clique duas vezes em um bloco para editar ou excluir.
  4. Submeta o fluxo para verificar se está correto.

Exemplos de Uso dos Blocos

Aqui estão exemplos de como os blocos podem ser usados para representar o algoritmo:

1. Definindo as Variáveis

Use o bloco Entrada para definir as variáveis:

num = 5
fatorial = 1
2. Representando Condições

Use o bloco Decisão para representar condições:

num > 1

Este bloco tem três conexões:

  • True: Quando a condição é verdadeira, o fluxo continua para realizar operações.
  • False: Quando a condição é falsa, o fluxo é direcionado para a Saída e encerra o processo.
  • Loop: Representa um retorno ao bloco de decisão para verificar novamente a condição. Isso é útil para repetir o processo enquanto a condição for verdadeira.

3. Realizando Operações

Use o bloco Processo para realizar operações:

fatorial = fatorial * num
num = num - 1

Após realizar as operações, o fluxo pode retornar ao bloco de Decisão (via Loop) para verificar se a condição ainda é verdadeira.

4. Representando a Saída de uma Variável

Use o bloco Saída para representar a referência de uma variável que contém um valor ou resultado:

fatorial

Desafio: Calcular o Fatorial de 5

Este desafio tem como objetivo treinar a lógica de programação através da visualização de processos usando fluxogramas. A proposta é resolver um problema matemático, estimulando o raciocínio lógico para estruturar uma solução passo a passo.

Imagine que você precisa calcular o fatorial do número 5, que é o produto de todos os números inteiros positivos de 1 até 5. Para isso, considere as seguintes informações:

  • Número inicial: 5
  • Variável do resultado (fatorial): Inicia em 1

Utilize o editor de fluxogramas para organizar o raciocínio e criar um fluxo que resolva este problema. Seu desafio é definir as etapas necessárias para:

  1. Inicializar as variáveis num e fatorial.
  2. Verificar se num é maior que 1.
  3. Se for verdadeiro, multiplicar fatorial por num e decrementar num em 1.
  4. Repetir o processo até que num seja igual a 1.
  5. Representar a saída do valor final de fatorial.
