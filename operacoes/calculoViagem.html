Ajuda - Como Criar Fluxos

Tipos de Blocos

  • Início: Ponto de partida do fluxo.
  • Entrada: Define variáveis e seus valores iniciais.
  • Processo: Realiza operações com as variáveis.
  • Saída: Representa a referência de uma variável que contém um valor ou resultado.
  • Fim: Ponto final do fluxo.

Como Criar um Fluxo

  1. Arraste os blocos da paleta para o editor.
  2. Conecte os blocos arrastando os pontos de conexão para definir a sequência lógica.
  3. Clique duas vezes em um bloco para editar ou excluir.
  4. Submeta o fluxo para verificar se está correto.

Exemplos de Uso dos Blocos

Os blocos abaixo são exemplos hipotéticos de como definir variáveis e realizar operações. Eles não representam necessariamente o fluxo do desafio atual.

1. Definindo Variáveis

Use o bloco Entrada para representar a definição de variáveis:

combustivel = 5.50
litros = 40
2. Realizando Operações

Use o bloco Processo para representar operações:

custoCombus = combustivel * litros
3. Representando a Saída de uma Variável

Use o bloco Saída para representar a referência de uma variável que contém um valor ou resultado:

custoCombus

Desafio: Calcular o Custo Total da Viagem

Este desafio tem como objetivo treinar a lógica de programação através da visualização de processos usando fluxogramas. A proposta é resolver um problema do dia a dia, estimulando o raciocínio lógico para estruturar uma solução passo a passo.

Imagine que você está planejando uma viagem e precisa calcular o custo total com base nas seguintes informações:

  • Custo do combustível por litro: R$ 5.50
  • Quantidade de combustível necessária: 40
  • Custo adicional com alimentação: R$ 120

Utilize o editor de fluxogramas para organizar o raciocínio e criar um fluxo que resolva este problema. Seu desafio é definir as etapas necessárias para:

  1. Calcular o custo total do combustível.
  2. Somar o custo total outros custos envolvidos na viagem com o combustível total gasto.
  3. Representar a saída do custo total da viagem.
