Ajuda - Como Criar Fluxos

Tipos de Blocos

  • Início: Ponto de partida do fluxo.
  • Entrada: Define variáveis e seus valores iniciais.
  • Decisão: Verifica uma condição e direciona o fluxo para Sim ou Não.
  • Processo: Realiza operações com as variáveis.
  • Saída: Representa a referência de uma variável ou resultado.
  • Fim: Ponto final do fluxo.

Como Criar um Fluxo

  1. Arraste os blocos da paleta para o editor.
  2. Conecte os blocos arrastando os pontos de conexão para definir a sequência lógica.
  3. Clique duas vezes em um bloco para editar ou excluir.
  4. Submeta o fluxo para verificar se está correto.

Objetivo do Desafio

O objetivo é criar um fluxo que resolva o problema de calcular a média de duas notas e verificar se o aluno foi aprovado ou reprovado. Para isso, você deve:

  1. Definir as notas do aluno.
  2. Calcular a média das notas.
  3. Verificar se a média é maior ou igual a 6.0.
  4. Representar o resultado final (Aprovado ou Reprovado).

Exemplos de Uso dos Blocos

Aqui estão exemplos de como os blocos podem ser usados para representar o algoritmo:

1. Definindo as Notas

Use o bloco Entrada para definir as notas:

nota1 = 7.9
nota2 = 8.4
2. Calculando a Média

Use o bloco Processo para calcular a média:

media = (nota1 + nota2) / 2
3. Verificando a Aprovação

Use o bloco Decisão para verificar a condição:

media >= 6.0

Este bloco tem duas conexões: Sim (aprovado) e Não (reprovado).

4. Representando o Resultado

Use o bloco Saída para representar o resultado final:

Aprovado
Reprovado

Isso indica o caminho que o fluxo deve seguir com base na condição.
