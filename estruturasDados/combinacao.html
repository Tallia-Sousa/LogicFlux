Ajuda - Como Criar Fluxos

Tipos de Blocos

  • Início: Ponto de partida do fluxo.
  • Entrada: Define variáveis e seus valores iniciais.
  • Processo: Realiza operações com as variáveis.
  • Saída: Representa a variável que contém o resultado final.
  • Fim: Ponto final do fluxo.

Como Criar um Fluxo

  1. Arraste os blocos da paleta para o editor.
  2. Conecte os blocos arrastando os pontos de conexão para definir a sequência lógica.
  3. Clique duas vezes em um bloco para editar ou excluir.
  4. Submeta o fluxo para verificar se está correto.
  5. Limpe o fluxo se necessário.

Exemplos Práticos

Aqui estão alguns exemplos de como usar os blocos para resolver o desafio de comparar preços de produtos em duas lojas:

1. Definindo as Listas de Preços

Use o bloco Entrada para definir as listas de preços das lojas A e B:

lojaA = [10, 30, 12]
lojaB = [9, 16, 8]

Isso define lojaA e lojaB com os preços dos produtos.

2. Unindo as Listas de Preços

Use o bloco Processo para unir as listas de preços:

precosCombinados = lojaA + lojaB

Isso cria uma nova lista chamada precosCombinados, que contém todos os preços das duas lojas.

4. Representando a Saída de uma Variável

Use o bloco Saída para representar a referência de uma variável que contém um valor ou resultado:

precosCombinados

Desafio: Comparar Preços de Produtos em Duas Lojas

Este desafio é focado em praticar a lógica de programação através de fluxogramas. Imagine que você está comparando preços de produtos entre duas lojas para encontrar a melhor opção de compra. As informações que você tem são as seguintes:

  • Preços na Loja A: [10,30, 12]
  • Preços na Loja B: [9, 16, 8]

Seu objetivo é criar um fluxograma que faça o seguinte:

  1. Receber as listas de preços dos produtos na Loja A e na Loja B.
  2. Unir as listas para visualizar todos os preços de ambos os locais.
  3. Representar a saída com a lista combinada.

Utilize o editor de fluxogramas para organizar o processo passo a passo.
