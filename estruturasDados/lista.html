Desafio: Somando Todos os Números em uma Lista

Este desafio é focado em praticar a lógica de programação através de fluxogramas. Você precisa somar todos os números de uma lista fornecida.

A lista de números é:

  • Lista: {8, 15, 3, 42, 10}
  • Tam da Lista: 4
  • inicializar contador: 0
  • Inicializar variavel de soma: 0

Seu objetivo é criar um fluxograma que faça o seguinte:

  1. Receber a lista de números.
  2. Inicializar uma variável soma com o valor 0.
  3. Somar cada número da lista à variável soma.
  4. Exibir a soma total ao final.

Utilize o editor de fluxogramas para organizar o processo passo a passo.
