Desafio: Calcular Média e Verificar Aprovação

Este desafio tem como objetivo treinar a lógica de programação através da visualização de processos usando fluxogramas. A proposta é resolver um problema do dia a dia, estimulando o raciocínio lógico para estruturar uma solução passo a passo.

Imagine que você precisa calcular a média de duas notas de um aluno e verificar se ele foi aprovado. Considere as seguintes informações:

  • Nota 1: 7.9
  • Nota 2: 8.4
  • Média necessária para aprovação: 6.0

Utilize o editor de fluxogramas para organizar o raciocínio e criar um fluxo que resolva este problema. Seu desafio é definir as etapas necessárias para:

  1. Receber as duas notas.
  2. Calcular a média das notas.
  3. Verificar se a média é maior ou igual a 6.0.
  4. Mostrar a saída informando se o aluno foi "Aprovado" ou "Reprovado".
